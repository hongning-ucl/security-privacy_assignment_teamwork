# 5-LEARNING-ON-GRAPHS

## Using GitHub
If this is your first time using GitHub, it would be useful to complete the _Git & GitHub Fundamentals_ assignment at https://classroom.github.com/a/c5Me39XH.
This assignment illustrates the basic functionalities of GitHub and how to work with them.

## Intended Learning Outcomes
* Create a graph from scratch using NetworkX and understand its representation as an adjacency matrix $A$ and node feature matrix $X$.
* Convert a NetworkX graph into a PyTorch Geometric `Data` object for use with graph learning frameworks.
* Train a baseline Multi-Layer Perceptron (MLP) on node features and understand its limitations when graph structure is ignored.
* Implement and train a Graph Convolutional Network (GCN) that exploits relational structure via message passing.
* Extract and visualise learned graph embeddings using PCA and t-SNE to understand what the models have learned.
* Explore community detection on a Stochastic Block Model (SBM) dataset to see why GNNs outperform MLPs when labels depend on graph structure.
* (Optional) Implement and compare alternative GNN architectures: Graph Attention Network (GAT) and Graph Isomorphism Network (GIN).
